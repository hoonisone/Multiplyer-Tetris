[문제]
다음과 같이 동작하는 프로그램을 작성한다.
결과 코드를 [문제이름]-[자기이름].txt로 본 디렉터리에 저장한다.

[동작]
문자 하나를 입력을 받고 입력된 시간을 출력하는 것을 무한 반복한다.
q를 입력했다면 입력시간을 마저 출력하고 프로그램을 종료한다.
출력되는 시간은 소수 셋째자리까지만 출력한다.

[예시]
-------- a 입력 ----------
a를 입력했습니다. (1.375초 경과)

-------- b 입력 ----------
a를 입력했습니다. (1.375초 경과)
b를 입력했습니다. (5.157초 경과)

-------- q 입력 ----------
a를 입력했습니다. (1.37초 경과)
b를 입력했습니다. (5.15초 경과)
q를 입력했습니다. (11.20초 경과)
프로그램을 종료합니다.

[Hint!]
* <time.h>에 대해 검색해보고 필요한 함수 사용
